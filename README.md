## redux 연습용 기본 코드

<br />

> ### 소개

- 리덕스의 작동 과정을 살펴보기 위해 구현한 기본적인 리덕스 예제 입니다.
- redux-actions, redux-thunk, redux-saga를 테스트 하기 위해 만들었습니다.
